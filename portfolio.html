Car Database

The purpose of this project was to scrape data from car listing sites, store the data in an SQL database and display details such as year, make, model, etc. in a table. The database is accessed by an MVC ASP.NET web app and also by a desktop WinForms app. The desktop app has a basic web scraper to extract JSON data.

Calculator

This calculator is an early portfolio project as part The Odin Project’s front end course. It was intended it to mimic a physical calculator. It does basic calculations: add, subtract, multiply and divide, and deals with negative numbers. It accepts keyboard inputs on desktop and scales down to mobile resolutions.
